##  3. ve 4. Hafta Çalışma Amacı
Bu iki haftalık süreçte amacım sadece araştırma yapmak değil; bir web projesinin
GitHub disiplini, backend (Spring Boot), frontend (React), içerik yönetimi (WordPress),
sosyal medya görünürlüğü ve SEO gibi tüm parçalarının birbiriyle nasıl bağlantılı
çalıştığını bütüncül olarak öğrenmek ve AIHEXA projesine nasıl katkı sağlayabileceğimi
anlamaktır.

##  Kullanılacak Teknolojiler
- Git & GitHub
- Java / Spring Boot
- React (JavaScript)
- WordPress
- Sosyal medya yönetim araçları (Instagram, LinkedIn)
- SEO analiz araçları

##  Öğrenilecek Konular
- Versiyon kontrol sistemleri ve GitHub iş akışı
- Backend API mantığı (Spring Boot, REST API, CRUD)
- Frontend component mantığı (React, props, state)
- WordPress ile özel yazılım arasındaki fark
- Sosyal medya stratejileri ve içerik planlama
- SEO temelleri ve içerik optimizasyonu
- AIHEXA modüllerinin analizi
